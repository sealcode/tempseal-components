# Quote component
Komponent zwracający tekst umieszczony w cudzysłowie z podpisem autora na dole.

### Przykład użycia

```javascript
await quote({ background, paragraph, autor });
```

### Argumenty

-   **background:** string, kolor tła

-   **paragraph:** string, tekst do wyświetlenia w cudzysłowie

-   **autor:** string, podpis
